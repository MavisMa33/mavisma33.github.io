HTML

HTML (HyperText Markup Language) is the most basic building block of the Web. It defines the meaning and structure of web content.

Table Tags and Attributes

Tag/Attributes Name/Format Description
<table> Table Enclose the content of the table.
<td> Table Data Table cell -- the smallest container inside a table.
<tr> Table Row Make rows to contain cells.
<th> Table Header Special cells that go at the start of a row or column and define the type of data that row or column contains.
<col> Column Specify styling information for each column.
<colgroup> Column Group Contain <col> element just under <table> tag.
<caption> Caption Contain a description of the table contents.
<thead> Table Head Structural definition to mark up a header section for the table.
<tbody> Table Body Structural definition to mark up a body section for the table.
<tfoot> Table Foot Structural definition to mark up a footer section for the table.
